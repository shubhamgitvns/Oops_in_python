## Oops

## Class: 
 class is a blueprint or templete for creating objects. it contains data members (attributes) and methods (functions).

 ## Object:
 An object is an instance of a class. It is a real entity created from a class that can access the attributes and methods.

 ## Attributes:
 Attributes are data stored inside an object.

 ## Constructer:
  constructor is a method or function . It runs automatically when a new object is created. 
  # in python __init__() is a constructer.

  ## Four Pillars of OOPs:
  1. Encapsulation - Wrapping the data and method inside the single unit (class).
  2. Abstraction   - Showing only needed data and hiding internal details.
  3. Inheritance   - Creating the new class from existing class (existing class is parent class and new class is child class).
  4. Polymorphism  - Same name, different behavior.
